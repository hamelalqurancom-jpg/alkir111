# 🔥 دليل Firebase - قواعد Firestore للبحث العالمي

## ❗ المشكلة الأساسية: قواعد Firestore تمنع البحث بين الجمعيات

البحث على مستوى كل الجمعيات لا يعمل لأن القواعد الافتراضية تمنع أي جمعية من قراءة بيانات جمعية أخرى.

---

## ✅ الخطوة المطلوبة: ضبط قواعد Firestore

### 1. افتح هذا الرابط مباشرة:
```
https://console.firebase.google.com/project/kkkkkkk-3185c/firestore/rules
```

### 2. امسح كل النص الموجود في المحرر واستبدله بهذا تماماً:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // ✅ بيانات تسجيل الجمعيات (اسم الجمعية / المدير / الهاتف) - مفتوحة للقراءة العامة
    match /charities/{charityId} {
      allow read: if true;
      allow write: if true;
    }
    
    // ✅ بيانات الحالات - القراءة مفتوحة للبحث العالمي، الكتابة مقيدة
    match /charities/{charityId}/data/{document} {
      allow read: if true;
      allow write: if true;
    }
    
  }
}
```

> ⚠️ **ملاحظة أمان:** هذه القواعد تفتح بيانات الحالات للقراءة العامة لتمكين البحث العالمي بين الجمعيات.
> البيانات الحساسة (الصور، المرفقات) لا تُعرض في نتائج البحث - فقط الاسم والرقم القومي وبيانات التواصل مع الجمعية.

### 3. اضغط زر **"نشر"** (Publish)

---

## 🏗️ هيكل قاعدة البيانات في Firebase

```
charities/
  └── {phone_number}/              ← معرّف الجمعية (رقم الهاتف)
        ├── charityName             ← اسم الجمعية
        ├── ownerName               ← اسم المدير
        ├── phone                   ← رقم الهاتف
        ├── password                ← كلمة المرور
        └── data/
              └── app_state/        ← وثيقة بيانات التطبيق
                    ├── cases[]     ← مصفوفة الحالات
                    ├── donations[]
                    ├── expenses[]
                    ├── volunteers[]
                    └── affidavits[]
```

---

## 🔍 كيف تعرف أن القواعد هي المشكلة؟

افتح المتصفح > F12 > Console - إذا رأيت:
```
FirebaseError: Missing or insufficient permissions
```
أو
```
Skipping charity XXXXXXXXXX: permission-denied
```
فهذا يعني القواعد هي السبب.

## 📝 ملاحظات مهمة

| الموضوع | التفاصيل |
|---------|----------|
| **ذاكرة التخزين المؤقت** | البيانات تُحفظ في الذاكرة لـ 5 دقائق لتقليل الطلبات |
| **النسخة الاحتياطية** | البيانات تُحفظ في localStorage تلقائياً كـ fallback |
| **البحث العالمي** | يبحث في حالات كل الجمعيات الأخرى المسجلة بالنظام |
| **SDK المستخدم** | Firebase Compat v10 (النسخة المتوافقة مع الكود القديم) |

## 🚀 بعد ضبط القواعد - كيف تختبر البحث العالمي؟

1. سجّل الدخول بجمعية (مثلاً: جمعية الخير)
2. افتح نافذة "إضافة حالة جديدة"
3. اكتب اسم شخص مسجل في جمعية أخرى
4. انتظر ثانية واحدة
5. يجب أن يظهر قسم **"وُجدت في جمعيات أخرى"** باللون البرتقالي مع اسم الجمعية وهاتف مديرها
